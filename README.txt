A module for phpVMS to view Vatsim Live Statistics

---------------------------------------------------

Install:

1. Download at Github.
2. Upload to your site in folder's order.
3. Access using the following code:
	<?php echo url('/vatsimstats') ;?>
4. Enjoy!
